Proyecto DevOps con GitHub

Este sitio forma parte de una práctica de control de versiones.

ACERCA DEL PROYECTO:

Este es un proyecto de Metodología DevOps

OBJETIVO DEL PROYECTO:

Este proyecto tiene como objetivo practicar el uso de Git y GitHub dentro de un flujo básico de DevOps. Esto permitirá a los practicantes a desarrollarse en conjunto para poder elaborar un mejor desempeño en el uso de GitHub y Git a través de VSC.

Tecnologías utilizadas para el proyecto

Este proyecto fue desarrollado utilizando HTML, CSS y GitHub como parte de una práctica de DevOps.
